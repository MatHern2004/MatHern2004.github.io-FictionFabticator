FictionFabricator

A domain-specific language intended to help create
immersive branching stories.

So, why not start crafting your own story?

Try it yourself! 

      
    name "Logan"
    personality "Logan
    item "Nothing" = true
    edit "Logan"
    setting boat
    choice one("Jump out of the boat?") empty "yes"
    {
    response "You jump out of the boat and drown, The End."
    }
